## Researched anchors (freelance, 2025–2026)
- **Web dev (RO):** Romanian freelance ≈ RON 70–82/h (~€14–16) employee-equivalent; freelance project rates run higher. RO ≈ 50% below US, 15–30% below German rates. *(Devico, PayScale)*
- **Graphic design / branding:** UK £25–100+/h (junior 25–40, mid 40–70, senior 70–150); global mid ≈ $45–85/h; logo £500–3,000; brand-identity tiers ≈ £750 / £1,500 / £2,500+. *(Wise, Inkbot, ManyPixels)*
- **Social media management:** freelance $500–2,500/mo retainer; hourly $35–150; tiers Basic $750–1,500, Standard $1,500–3,000, Premium $3,000–7,000+. *(SolidGigs, HawkSEM, Cloud Campaign)*
- **Copywriting:** $0.10–0.25/word (beginner) up to $0.50–1.00 (experienced); EU experienced ≈ €0.35–0.40/word, €85–100/h; hourly junior $50–85, mid $85–160, specialist $200–300. *(Ruul, Nomad, Mediabistro)*
- **Video editing:** hourly entry $20–35, mid $40–80, senior $100–150+; per finished minute $50–150; social clips $100–500; rush +25–50%. *(Creativize, Twine, Pixflow)*

## Positioning angles (pick 2–3 per deal)
Eastern-EU quality at competitive cost · solo = direct senior work, no agency overhead · multi-discipline under one roof (design + copy + web + video + AI) · fast turnaround · strong portfolio fit · AI-accelerated delivery.

## Pricing-to-market note
RO clients anchor on the RO market; EU/US clients anchor on their home market. **Price to the client's market, not to Fri's cost base.** That gap is margin — fairly earned, because the value delivered matches the client's market.

If you used only these static anchors, mark confidence "medium" and recommend a live check for anything above €2,000 or any unfamiliar niche.
