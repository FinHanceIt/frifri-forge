# Changelog

All notable changes to IdeaForge are documented here.

## [1.0.0] — 2026-06-17
Initial release.

- 11-agent think-tank: Facilitator + Philosopher, Explorer, Synthesist, Investigator, Advocate, Realist, Skeptic, Steward, Futurist, Strategist.
- Five session modes: Spark, Crucible, Horizon, Verdict, Roundtable.
- Shared Idea Dossier with a named section per thinker; two human gates (framing, verdict).
- Honesty charter: the one-fact rule (only the Investigator asserts external facts, with sources), confidence labels, steelman-before-critique, a dissent log.
- Anti-groupthink flow (diverge before converge; the Skeptic must attack the leading idea).
- Domain-agnostic by design; safety gate held by the Steward.
- Optional AppFactory-ready build brief when a software build is the chosen next step.
- Bilingual RO/EN (reason in English, deliver in the user's language).
- References: house-style, handoff-contracts, thinking-methods.
